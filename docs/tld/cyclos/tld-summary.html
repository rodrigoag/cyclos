cyclos

Standard Syntax:
     <%@ taglib prefix="cyclos" uri="http://devel.cyclos.org/tlibs/cyclos-core" %>
XML Syntax:
     <anyxmlelement xmlns:cyclos="http://devel.cyclos.org/tlibs/cyclos-core" />
 
Cyclos Core Tag Library

Tag Library Information
Display NameNone
Version1.1
Short Namecyclos
URIhttp://devel.cyclos.org/tlibs/cyclos-core
 

Tag Summary
noCacheSets the no-cache response headers.
escapeJSEscapes a String to JavaScript, either writing it to the response or exporting it to a scoped variable. The string may be the value attribute or the tag body.
escapeHTMLEscapes a String to HTML, either writing it to the response or exporting it to a scoped variable. The string may be the value attribute or the tag body.
customizedFilePathRenders a customized file path, or exports it to the given variable, if the var attribute is specified. When a user is logged, first check if the given file is customized for his specific group. If not, use the main customized file. Alternatively, a group or group filter may be specified using the groupId or groupFilterId attributes.
includeCustomizedFileIncludes a customized file on the current response. When a user is logged, first check if the given file is customized for his specific group. If not, use the main customized file. Alternatively, a group or group filter may be specified using the groupId or groupFilterId attributes.
formatFormats a given data according to the application's settings. One of the following attributes should be set: number, amount, date, dateTime or bytes. When none of those attributes were set, or the specified one was null, the default attribute is used to determine the output.
customFieldRenders a custom field. This tag can be used just output a custom field value (when the textOnly attribute) is set to true or to render a HTML control for that field.
menuRenders a menu item. May be nested, however, currently, only 2 levels are supported.
profileRenders element information like name and username. If the logged user has the required permissions a link to the profile is shown.
helpRenders the help icon, inside or not a TD element
imagesRenders an image index, allowing them to be edited
layoutThis tag generates a HTML table element with a fixed column count. As nested cell tags are used, they fill in those columns. When the column count reaches the table's column limit, a row break is generated. The row break may also be forces with a rowBreak tag. Arbitrary attributes may be passed to the table tag, with the following replacements: class should be specified as className, and id as elementId.
cellThis tag generates a HTML cell inside a layout tag. Arbitrary attributes may be passed to the table tag, with the following replacements: class should be specified as className, and id as elementId.
rowBreakForces a row break inside a layout tag
multiDropDownRenders a multi drop down control. A multi drop down is a control generated via JavaScript that is like a HTML multiple select, but is expandable, having a checkbox on each option.
optionRenders a multi drop down option
paginationRenders a pagination index for a search results. When the user clicks a page number, a HTML hidden is created on the first form, with the name 'query(pageParameters).currentPage'. This attribute matches the page parameters for actions extending nl.strohalm.cyclos.controls.BaseQueryAction. For an alternative form, the form attribute may be used, indicating the form name.
truncateTruncates a string for a given number of lines, with a given max length each.
customImageUrlRenders the url for a given custom image
customImageRenders an img tag for a given custom image. Arbitrary attributes may be passed to the img tag, with the following replacements: class should be specified as className, and id as elementId.
richTextAreaRenders a textarea with options to select if the content is plain text or rich text
selectCustomFieldsGiven a collection of custom fields, select those who matches specified conditions, storing the resulting collection on the page
scriptRenders a script tag, appending the cyclos version as a parameter to avoid browser cache between versions
clearCacheInvokes the CacheCleaner.clearCache() method
 

Function Summary
booleangranted( nl.strohalm.cyclos.access.Permission)Returns if the logged user has granted the given permission
Stringname( java.lang.Object)Given an enumeration instance, returns it's name
Objectvalue( java.lang.Object)Given an enumeration instance, returns it's value. The value is determined using the following heuristic: If the enum implements nl.strohalm.cyclos.utils.StringValuedEnum or nl.strohalm.cyclos.utils.IntValuedEnum the getValue() method is called, otherwise, the enum name is returned.
booleancontains( java.lang.Object, java.lang.Object)Returns if a collection, array or map contains a given value (or key if a map)
java.math.BigDecimalround( java.lang.Object)Rounds a given number according to the local settings
booleanequals( java.lang.Object, java.lang.Object)If both objects are non-null then check for equality (using the equals'' implementation of the first argument). if any of them is empty then return false.
